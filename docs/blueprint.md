# **App Name**: DevFlow

## Core Features:

- Quick Capture: Single textarea for quick entry of notes, code snippets, and ideas.
- Auto-Tagging: AI automatically categorizes entries based on content (e.g., code_snippet, learning_note, idea, bug_fix, general) using a machine learning tool.
- Smart Search: Enable natural language search to quickly find entries based on queries.
- Timestamps: Automatically timestamp entries for easy tracking.

## Style Guidelines:

- Primary color: Soft blue (#73A5EB) for a calm, focused feel, suitable for long work sessions.
- Background color: Light gray (#F5F7FA), a desaturated version of the primary color, for minimal distraction.
- Accent color: Warm orange (#EBA973), analogous to blue, to highlight important actions and tags, creating contrast.
- Body and headline font: 'Inter', a grotesque sans-serif known for its readability, suitable for both headlines and body text.
- Use simple, monochromatic icons to represent categories and actions.
- Clean, minimalist layout with a focus on readability and quick access to the input box.
- Subtle animations for feedback, such as saving confirmation.
